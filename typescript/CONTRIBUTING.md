# Contributing to ashid

Thanks for your interest in contributing to ashid!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ashid.git`
3. Install dependencies: `pnpm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
pnpm build    # Build the library
pnpm test     # Run tests
pnpm lint     # Lint code
```

## Pull Request Process

1. Ensure tests pass and coverage is maintained
2. Update README.md if adding new features
3. Follow existing code style
4. Write clear commit messages

## Reporting Bugs

Please use the [bug report template](https://github.com/wildeagency/ashid/issues/new?template=bug_report.md) when reporting issues.

## Feature Requests

Please use the [feature request template](https://github.com/wildeagency/ashid/issues/new?template=feature_request.md) for new ideas.

## Code of Conduct

Be respectful. Be constructive. We're all here to build something useful.
